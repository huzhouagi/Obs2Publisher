# V1.0.0

Obs2Publisher 首个公开发行版本。

- 支持微信公众号、X Articles 和小红书三个内容平台，在 Obsidian 中实时预览当前 Markdown 笔记。
- 微信公众号支持 8 套内联样式主题、5 种字体、富文本复制、后台快捷入口和 HTML 导出。
- X Articles 支持标题与正文复制、图片占位、表格高清截图、文章图片导出和后台快捷入口。
- 小红书支持 5 套 3:4 卡片主题、章节分页、内容概要编辑，以及单页或批量导出 1080x1440 PNG 图片。
- 支持 Vault 本地图片、相对路径图片、Wiki 图片、Base64 图片和 HTTP/HTTPS 外链图片。
- 支持分别设置三个平台的 Vault 内导出目录，并在导出后返回实际文件路径。
- 内容预览底部提供“关于作者”和“设置”入口，可查看胡诌简介、社交账号及微信和支付宝赞赏码。
- 所有内容转换均在本地完成，不调用 AI，不上传笔记，不收集遥测。
